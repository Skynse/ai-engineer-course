# Lesson 13: Final Project Planning

**Duration**: 30 minutes  
**Goal**: Pick a project and write the Codex prompt that kicks it off

## Pick an Idea

Choose something you'd actually use. It needs:

- user authentication
- at least two related collections
- at least one file upload

If you can't think of something, here are three starting points: a book tracker, an event planner, or a study notes app.

## Scope It Down

Write one sentence describing what the app does. Then list the three features that absolutely must work for the app to be useful. Everything else is a nice-to-have.

## Write Your Build Prompt

Before the next session, write the prompt you'll give Codex to scaffold the project. It should cover:

- what the app does
- who it's for
- the core features
- the stack (Next.js + Appwrite)
- what not to build yet

A tight prompt here saves hours later.
