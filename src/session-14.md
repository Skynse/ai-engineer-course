# Lesson 14: Final Project Building

**Duration**: ongoing  
**Goal**: Build your project one working slice at a time

## The Only Workflow You Need

1. Pick one feature small enough to finish in a single session
2. Write the expected outcome before you prompt Codex
3. Ask Codex to implement it
4. Verify it works before moving on

Don't ask Codex for the whole app at once. One slice at a time.

## When You're Stuck

Paste the error into Codex with this context:

```
Error: [paste error]
Expected behavior: [what should happen]
What changed: [brief note]

Explain the likely cause first, then patch the minimum code needed.
```

## Before You Submit

- [ ] All core features work end-to-end
- [ ] App is deployed and publicly accessible
- [ ] You can demo it without it breaking

---

**You're done. Ship it.**
